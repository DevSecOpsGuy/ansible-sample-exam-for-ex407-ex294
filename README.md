# **Ansible Exam for EX407/EX294**

Provides machines configuration

## **Prerequisites**

- [Vagrant](https://www.vagrantup.com/)
- [Virtualbox](https://www.virtualbox.org/)

## **Playground**

- Generate ssh key pair

```sh
ssh-keygen -f ./ssh-keys/ansible_exam -N "" -q
```

- Run `vagrant`

```sh
vagrant up
```
